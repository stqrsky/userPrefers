ACTUAL USER PREFERS DARK MODE?

When it comes to innovative UX/UI the UX-Persona group of "gamer" commonly count to the "early adopters" of new techniques and styles. An DarkMode UI is probably one of the widely spreaded example for this thesis and at least since windows and macOs are providing an darkmode option able to choose for the user as root, the theme mode enjoys great popularity among all classes of UX-Persona read more

    IN ORDER TO DETECT AND TOGGLE USERS PREFERENCES BY OS ROOT WE WILL:

  • 1. detect the selected color-scheme for root OS by using the "prefers-color-scheme" level5 CSS3 media-querie
  • 2. add a custom css :root class ".darkmode" using the JAVASCRIPT method window.matchMedia("(prefers-color-scheme: dark)").matches to fire whenever our media-querie detects the color-scheme: dark
  • Note: It lays on the hand that we we'll setup the "light color scheme" as css :root basic in order to make described process work
